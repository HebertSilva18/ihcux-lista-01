Cantina do Caos

Aplicação console desenvolvida em C# (.NET) como exercício acadêmico.


Sobre o projeto

O programa simula um sistema de pedidos de uma cantina universitária,
onde o usuário pode escolher um produto do menu, informar a quantidade
e confirmar o pedido — tudo pelo terminal.

Heurísticas de Nielsen aplicadas: 

**Heurística #1 — Visibilidade do Status**
O sistema informa em qual etapa o usuário está durante todo o fluxo,
exibindo [Passo 1 de 3], [Passo 2 de 3] e [Passo 3 de 3].

**Heurística #3 — Controle e Liberdade**
O usuário pode digitar 'voltar' para corrigir a etapa anterior
ou 'cancelar' para encerrar o pedido a qualquer momento.

**Heurística #9 — Ajuda e Erros**
As mensagens de erro são claras e orientam o usuário, informando
o código digitado e os códigos válidos disponíveis.

 Como executar:

1. Ter o .NET instalado na máquina
2. Abrir o terminal na pasta do projeto
3. Executar o comando: dotnet run

 Tecnologias:

- C# (.NET)
- Visual Studio Code
